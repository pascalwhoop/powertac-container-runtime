Before running:
You need to have R installed
https://www.r-project.org/

Then you need to install Rserve.
To do this run R, execute the following command and follow the instructions:
install.packages("Rserve")

Now you can run Rserve like this:
library("Rserve")
Rserve() 

Finally you can run CrocodileAgent with this command in terminal:
java -jar CrocodileAgent.jar
